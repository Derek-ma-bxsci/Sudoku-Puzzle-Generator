# Sudoku-Puzzle-Generator
Created by Derek Ma

This project lets you play a sudoku game with different difficulty levels or see a solved and valid sudoku board.

To run the project, use this JDoodle link:
https://www.jdoodle.com/ga/yCUsHDIunDvXFdAaxT4ZlA%3D%3D 

The sudoku board is generated by shuffling a list of numbers from 1 to 9; checking to see whether or not that number already appears in the row, column, or box; and if it is not already present places that number there. Then the code is repeated until the entire board is created.

Files included:
 - SudokuGenerator.java (main)
 - SudokuBoard.java
 - Solvableboard.java
